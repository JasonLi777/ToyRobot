How to run:
    double click ToyRobot.exe to start Toy Robot.

How to input commands from file:
    enter 'ToyRobot.exe command.txt' in cmd.exe to run Toy Robot.
    or use StartToyRobot_fileAsInput.bat to run ToyRobot and take command.txt as input.

How to view unit test results:
    Click UnitTest.bat to view unit test results.
